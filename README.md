# Офисный бот-помощник (Telegram)

Простой Telegram-бот для создания напоминаний.  
Пользователь отправляет сообщение в формате `Завтра 10:30 Позвонить клиенту` или `25.03.2025 14:00 Отчёт`, бот сохраняет задачу и присылает уведомление в назначенное время.

## Функционал

- Создание напоминаний (дата и время в свободной форме: `завтра 15:00` или `ДД.ММ.ГГГГ ЧЧ:ММ`).
- Просмотр текущих задач (`/tasks`).
- Отметка задачи выполненной (`/done <номер>`).
- Автоматическая отправка уведомлений.
- Логирование всех событий в файл `bot.log`.

## Технологии

- Python 3.10+
- `python-telegram-bot` (асинхронная версия)
- `asyncpg` (работа с PostgreSQL)
- `PostgreSQL` (база данных)
